# slim3-api-skeleton
My Slim Framework 3 API Skeleton with Logging

## Install

```shell
$ composer require slim/slim "^3.0"

$ composer require silalahi/slim-logger --dev dev-master

$ composer dump-autoload -o

```

create logs folder
